# 📊 Customer Churn Analytics Platform

A machine learning-powered web application to predict customer churn and provide business insights.

## 🚀 Features
- 🔍 Single customer prediction
- 📊 Bulk dataset analysis
- 📈 Risk segmentation (High / Medium / Low)
- 📉 Interactive dashboard with charts
- 🌐 Deployed using Streamlit Cloud

## 🛠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- Matplotlib

## 🌍 Live Demo
https://customer-churn-ai-8rhjt9izreak74bj2hdmsa.streamlit.app

## 📌 How it works
- Model trained on telecom churn dataset
- Predicts probability of churn
- Provides insights for business decisions
